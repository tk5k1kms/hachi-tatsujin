# 蜂の達人 HP デザイン仕様書
**作成日：2026年4月9日**
**参考サイト：https://hachi-pro.co.jp/**

---

## 1. 全体方針

ハチプロのデザインテイストを踏襲しつつ、蜂の達人オリジナルのブランディングを行う。
ハチプロの特徴は「画像バナー＋シンプルな情報設計＋オレンジの温かみ」。

### ハチプロとの決定的な違い（現在の蜂の達人HPの問題点）
1. **ヒーローが「CSSテキスト」になっている** → ハチプロは**デザインされた画像バナー**
2. **色味が暗すぎる** → ハチプロは**白背景＋オレンジ＋薄いベージュ**の温かみのある配色
3. **写真が1枚しかない** → ハチプロは駆除業者の実写写真を大きく使用
4. **レイアウトが詰まりすぎ** → ハチプロは余白が多くゆとりがある
5. **キャンペーンバナーのデザイン差** → ハチプロは紙風のカードデザイン

## 3. ヘッダー

### ハチプロのヘッダー構成
- 背景：白
- 左：蜂のアイコン＋「ハチプロ」大きなロゴ文字
- 右：電話アイコン ＋ 「受付時間 365日年中無休 0:00~24:00」＋ 電話番号（大きく太い文字）
- ヘッダーの下に影は最小限
- **固定ではない（スクロールで消える）**

### 蜂の達人の実装
- 背景：白、下線にオレンジのボーダー（3px）
- 左：ロゴ画像（logo.png）＋「蜂の達人」テキスト
- 右：受付時間ボックス ＋ 「蜂駆除にお困りの方はこちらから」＋ 電話番号
- 固定ヘッダー（position: fixed）

---

## 4. ヒーロー（メインビジュアル）

### ハチプロのヒーロー
**これが最大の差別化ポイント。ハチプロのヒーローは「デザインされた画像」。**

- 背景：薄いベージュ/クリーム色（暗い写真ではない）
- 右側：防護服を着た駆除業者の写真（切り抜き）
- 左側にテキスト要素（画像の一部として配置）：
  - 「ハチに刺される前に」（白背景のラベル）
  - 「最短30分到着」（金色バッジ）
  - 「プロが速攻！」（大きな太文字）
  - 「ハチを徹底駆除！」（超大きな太文字、赤い強調）
  - 金メダルバッジ4つ（安全・確実・迅速・早い）
- 右下：「対応エリア」緑のボタン

### 蜂の達人の実装方針
**Gemini（Nano Banana）で以下の要素を含む1枚の画像バナーを生成し、それをヒーローに使用する。**

生成すべき画像：
1. 薄いベージュ/クリーム背景
2. 右側に防護服の駆除業者（hero-bg.pngを活用）
3. 左側にテキスト：「ハチに刺される前に」「プロが速攻！蜂を徹底駆除！」
4. 金メダルバッジ4つ
5. 「最短即日対応」バッジ

**→ Pythonで画像を合成して作成するか、Geminiで1枚画像として生成する。**

## 5. キャンペーンバナー

### ハチプロの構成
- 背景：鮮やかなオレンジ〜赤のグラデーション
- 左：「初めての方限定」＋「LINE限定キャンペーン」
- 中央：「今月限定」黄色丸バッジ
- 右：「今LINEからお申し込みいただくと 2,000円OFF」（2,000は超大きく赤文字）
- LINEボタン＋QRコード
- 全体的にカード的なデザイン（角丸、影あり）

### 蜂の達人の実装
- ほぼ同じ構成で実装
- QRコードは秘書フォルダにあるものを使用可能

## 6. 固定CTA（画面下部）

### ハチプロの構成
- 緑色の大きな角丸ボタン：「電話で 今すぐ相談 をする ▶」
- 受付時間・電話番号も表示
- LINEボタンのQRコード付き

### 蜂の達人の実装
- 2分割：LINE緑ボタン ＋ 電話ボタン
- 電話ボタンはハチプロ風の緑にする
